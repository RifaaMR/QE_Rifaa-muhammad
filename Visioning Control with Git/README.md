Rangkuman
Versioning merupakan proses untuk mengatur versi dari source code program.
a. Tools untuk versioning :

- Version control system (VCS)
- Source code manager (SCM)
- Revision control system (RCS)

b. GIT : merupakan salah satu version control system popular yang digunakan oleh para developer untuk membangun software secara Bersama-sama

c. The staging area : dalam working directory masukan “git add” untuk masuk staging area, kemudian dari staging area masukkan “git commit” untuk disimpan ke git repository

d. GIT Branching : Git branching adalah konsep di Git yang memungkinkan pengembang untuk bekerja secara paralel pada berbagai fitur atau perbaikan dalam proyek perangkat lunak. Saat menggunakan Git branching, pengembang dapat membuat cabang (branch) baru dari titik tertentu dalam sejarah repositori, yang memungkinkan mereka untuk memisahkan dan mengisolasi perubahan yang berbeda tanpa mengganggu cabang utama (main atau master).
